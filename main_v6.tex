\ProjHead{Ehanced Lubrication in Superhydrophobic Microchannels}{NSF award 2140033}
      \ProjListII
      \ProjBullet{Designed and fabricated a transverse grooved superhydrophobic microchannel using SU-8 Soft-lithography with PDMS in a class 1000 cleanroom (Tufts Microfab) and examined the flow-field using the micro-PIV technique.}

      \ProjListII
      \ProjBullet{Improved the existing direct permeability measurement technique for aerogel. Utilized Duhamel’s theorem, and developed a non-destructive inverse approach to measure aerogel permeability.}
      \ProjBullet{Developed an experimental apparatus for precise aerogel permeability measurement under supercritical CO$_2$ flows, and several custom LabVIEW and MATLAB programs for data collection and post-processing.}  
      \ProjListIIEnd

\ProjHead{Jet-impingement enhanced freeze drying of aerogel}{NSF STTR Phase I award 2014881}
      \ProjListII
      \ProjBullet{Designed, constructed, and utilized a first-of-its-kind drying system demo using jet-impingement to optimize Aerogel Technologies, LLC.'s patented freeze-drying process, resulting in a 75\% reduction in production time.}

      \ProjListIIEnd
